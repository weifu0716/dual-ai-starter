# 雙 AI 協作起手包（給人看的版本）

你朋友幫你準備了一套「兩個 AI 互相配合幹活」的安裝手冊。你不用懂技術，照三步走：

## 第一步
打開你已經有的那個 AI（Claude 或 ChatGPT 都行，用網頁版或 App 都可以）。

## 第二步
把 `雙AI協作起手包_SETUP.md` 這個檔案丟給它（或整段複製貼上），然後打這句話：

> 照這份手冊帶我一步一步做，我做完你檢查，過了才走下一步。

## 第三步
它說什麼你做什麼。過程中它會帶你：
- 訂閱缺的那一邊（先買入門級就好）
- 裝好兩個 AI 的工具
- 蓋好它們共用的規矩
- 最後跑一個小實驗：你只講需求，兩個 AI 自己分工把東西做出來給你驗收

全程大概一個下午。卡住就把錯誤訊息拍照丟回去問它。
